# Análisis de Recuperación del Turismo en Honduras (2019-2023)

Este proyecto analiza el impacto del COVID-19 en el sector turístico de Honduras y su posterior recuperación, integrando un flujo completo de datos.

## Tecnologías Utilizadas
* **Python (Pandas)**: Proceso ETL, limpieza y generación de datos.
* **SQL (SQLite)**: Almacenamiento y consultas analíticas.
* **Power BI**: Dashboard interactivo con indicadores clave de recuperación.

##  Estructura del Proyecto
* `datos_etl.ipynb`: Notebook con el proceso de limpieza.
* `data/`: Contiene la base de datos `.db` y el `.csv` para Power BI.
* `analisis_turismo.sql`: Scripts SQL con consultas de negocio.
* `Informe_turismo_Honduras.pbix`: Archivo fuente del dashboard.
